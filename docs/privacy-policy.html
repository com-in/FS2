文件保护助手隐私政策

生效日期：2026年01月10日

1. 引言

CMCS Studio（以下简称"我们"）尊重并保护所有使用文件保护助手（以下简称"本软件"）用户的个人隐私。本隐私政策说明了我们如何处理您的信息，以及我们对您隐私的承诺。

2. 数据收集

本软件在使用过程中，不会收集、存储任何您的个人数据或文件内容。所有文件保护操作均在您的本地设备上完成。

唯一的例外情况是：当您主动使用"检查更新"功能时，本软件会连接到GitHub服务器以获取最新版本信息。在此过程中，会传输匿名的版本检查请求，不包含任何个人身份信息或文件内容。

3. 数据使用

由于我们不收集任何个人数据，因此不存在数据使用的情况。本软件的所有核心功能均在您的设备本地运行，无需访问或使用您的任何个人信息。

当您使用"检查更新"功能时，传输的匿名请求仅用于获取软件更新信息，不会被用于其他任何目的。

4. 数据共享

我们不会分发、共享、出售或转让您的任何个人数据给第三方。

当您使用"检查更新"功能时，仅会与GitHub服务器进行匿名的数据传输，用于获取软件更新信息。

5. 数据安全

本软件的核心功能不收集或存储任何个人数据，因此不存在相关的数据泄露或安全风险。您的所有文件和信息始终保留在您自己的设备上，由您完全控制。

当您使用"检查更新"功能时，与GitHub服务器的连接采用加密传输，确保数据传输的安全性。传输的内容仅为匿名的版本检查请求，不包含任何敏感信息。

6. 政策更新

我们可能会不时更新本隐私政策。任何更新都会通过软件更新或我们的官方渠道通知您，更新后的政策将在生效日期后对所有用户生效。

File Protection Assistant Privacy Policy

Effective Date: January 10, 2026

1. Introduction

CMCS Studio (hereinafter referred to as "we") respects and protects the privacy of all users of File Protection Assistant (hereinafter referred to as "this software"). This privacy policy explains how we handle your information and our commitment to your privacy.

2. Data Collection

This software does not collect or store any of your personal data or file content during use. All file protection operations are completed locally on your device.

The only exception is: when you actively use the "Check for Updates" feature, this software will connect to GitHub servers to obtain the latest version information. During this process, an anonymous version check request is transmitted, which does not contain any personal identification information or file content.

3. Data Usage

Since we do not collect any personal data, there is no data usage situation. All core functions of this software run locally on your device and do not require access to or use of any of your personal information.

When you use the "Check for Updates" feature, the anonymous requests transmitted are only used to obtain software update information and will not be used for any other purpose.

4. Data Sharing

We will not distribute, share, sell, or transfer any of your personal data to third parties.

When you use the "Check for Updates" feature, only anonymous data transmission is performed with GitHub servers for the purpose of obtaining software update information.

5. Data Security

The core functions of this software do not collect or store any personal data, so there are no related data leakage or security risks. All your files and information are always kept on your own device and fully controlled by you.

When you use the "Check for Updates" feature, the connection to GitHub servers uses encrypted transmission to ensure the security of data transmission. The transmitted content is only an anonymous version check request and does not contain any sensitive information.

6. Policy Updates

We may update this privacy policy from time to time. Any updates will be notified to you through software updates or our official channels, and the updated policy will take effect for all users after the effective date.
