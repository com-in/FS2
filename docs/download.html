使用说明 Usage Instructions

文件保护助手的使用非常简单： Using File Protection Assistant is very simple:

  1. 打开文件保护助手应用程序 Open File Protection Assistant application
  2. 选择您想要加密或解密的文件 Select the file you want to encrypt or decrypt
  3. 输入您的密钥（请确保记住这个密钥，它是解密文件的唯一方式） Enter your key (please remember this key, it is the only way to decrypt the file)
  4. 点击"加密"或"解密"按钮 Click the "Encrypt" or "Decrypt" button
  5. 等待操作完成，您将得到加密或解密后的文件 Wait for the operation to complete, you will get the encrypted or decrypted file
